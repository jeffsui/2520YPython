# Python学习课程代码

这是一个全面的Python基础学习项目，包含了12天的学习内容，涵盖了Python语言的核心概念和实用技术。

## 项目结构

项目按照学习天数分为12个目录，从基础语法到高级特性循序渐进：

### Day 1: Python基础语法
- 基本输入输出
- 数据类型：str, int, float, bool
- 运算符：+, -, *, /, //, %, **
- 类型转换：int(), float(), str(), bool()
- 进制转换：bin(), oct(), hex()
- ASCII码转换：ord()

### Day 2: 条件判断
- if条件语句
- if-else结构
- if-elif-else多重分支
- 逻辑运算符：and, or, not
- 成绩分级实例

### Day 3: 循环结构
- while循环基础
- for-range循环
- break和continue控制
- 循环嵌套
- 数字统计与求和

### Day 4: 列表(List)
- 列表创建与访问
- 列表索引与切片
- 列表操作：增、删、改、查
- 列表遍历方法
- 列表与for循环结合

### Day 5: 高级数据结构
- 列表进阶操作
- 元组(Tuple)使用
- 排序算法：冒泡排序、选择排序
- 数据结构对比与选择

### Day 6: 字典与集合
- 字典(Dict)创建与访问
- 字典操作：增、删、改、查
- 字典遍历方法
- 集合(Set)使用
- 数据结构去重

### Day 7: 字符串处理
- 字符串基本操作
- 字符串方法与函数
- 字符串格式化
- 字符串切片与拼接
- 实用字符串处理技巧

### Day 8: 函数基础
- 函数定义与调用
- 参数传递：位置参数、默认参数
- 关键字参数
- 可变参数：*args和**kwargs
- 函数返回值
- 作用域规则

### Day 9: 高级函数特性
- 函数作为参数
- Lambda表达式
- 高阶函数
- 闭包概念
- 装饰器基础
- 变量作用域详解

### Day 10: 常用模块
- 时间模块：time, datetime
- 操作系统模块：os
- 网络请求：requests
- 递归函数
- 自定义模块与导入
- HTML内容获取与处理

### Day 11: 文件操作与异常处理
- 文件读写操作：read, readline, readlines
- 文件打开模式：r, w, a, r+
- 异常处理：try-except-else-finally
- 常见异常类型
- 文件上下文管理：with语句
- CSV文件处理
- 生成器基础

### Day 12: 面向对象编程
- 类与对象概念
- 属性与方法
- 构造函数与析构函数
- 私有属性与公有属性
- 类方法与静态方法
- 继承基础
- 随机数模块random
- 封装、继承、多态初探

## 特色亮点

1. **实践性强**：每个知识点都有对应的代码示例
2. **循序渐进**：从基础到高级，难度递增
3. **涵盖全面**：包括基础语法、数据结构、算法、函数、模块、面向对象等核心内容
4. **注释详细**：代码中包含大量注释，便于理解
5. **场景丰富**：融入游戏修仙等有趣元素，提高学习兴趣

## 学习建议

1. 按照天数顺序学习，每天集中掌握一个主题
2. 动手实践每个代码示例，尝试修改和扩展
3. 理解每个概念的原理，而不仅仅是记忆语法
4. 结合实际项目应用所学知识
5. 定期复习前面的内容，加深理解

## 运行环境

- Python 3.x
- 推荐使用VS Code或PyCharm等IDE
- 部分示例需要安装requests等第三方库
